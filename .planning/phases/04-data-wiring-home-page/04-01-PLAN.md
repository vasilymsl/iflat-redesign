  <action>
Создать файл `iflat-redesign/src/lib/tv-data.ts` с двумя exported async-функциями:

1. **`fetchChannels(): Promise<ChannelData[]>`**
   - Получает guest token через `getGuestToken()` из `@/lib/tv-token`
   - Вызывает `GET {TV_API_BASE}/v2/channels` с `Authorization: Bearer {token}` и `next: { revalidate: 300 }` (кеш 5 минут)
   - При успешном ответе маппит данные через `mapApiChannels()` в `ChannelData[]`
   - При любой ошибке (non-ok status, network error, маппинг вернул пустой массив) — возвращает `freeChannels` из `@/config/tv-shelves` как fallback
   - Логирует первые 300 символов ответа через `console.log("[tv-data] channels raw sample:", ...)` для будущей отладки формата

2. **`fetchNovinki(): Promise<ContentItem[]>`**
   - Аналогично: guest token, `GET {TV_API_BASE}/v2/rows/novinki`, `revalidate: 300`
   - Маппинг через `mapApiNovinki()` в `ContentItem[]`
   - Fallback: `newReleases` из `@/config/tv-shelves`
   - Логирование первых 300 символов

3. **`mapApiChannels(data: unknown): ChannelData[]`** (internal)
   - Защитный маппинг: пробует `data.channels`, `data.data`, или `data` если массив
   - Каждый item маппится в `ChannelData`: `id` (string), `name`, `logo` (пробует logo_url/logo/icon), `currentProgram` (пробует current_program.title/current_program), `thumbnail` (пробует thumbnail_url/preview_url/thumbnail), `progress`
   - Фильтрует items без id или name
   - Обернуто в try/catch — при ошибке возвращает `[]`

4. **`mapApiNovinki(data: unknown): ContentItem[]`** (internal)
   - Защитный маппинг: пробует `data.items`, `data.content`, или `data` если массив
   - Маппинг в `ContentItem`: `id`, `title` (name/title), `subtitle` (year + genre), `poster` (poster_url/poster/thumbnail), `rating`, `href`
   - Фильтрует items без id/title/poster
   - Обернуто в try/catch

Константа `TV_API_BASE` = `process.env.TV_API_BASE_URL ?? "https://api.24h.tv"` (аналогично tv-token.ts).

Типы импортировать из существующих файлов:
- `import type { ChannelData } from "@/components/sections/ChannelCard"`
- `import type { ContentItem } from "@/components/sections/ContentShelf"`

Полный пример кода есть в 04-RESEARCH.md секция "Code Examples / Полная tv-data.ts с fallback" — использовать его как основу.
